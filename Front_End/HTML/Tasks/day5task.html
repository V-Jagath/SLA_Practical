Header

Header is a semantic tag used to define the top part (head portion) of a webpage or a section.

Navication

Navication is a semantic tag used to define navigation links in a webpage.

Main

Main is a semantic tag used to define the main content of the webpage.

Article

Article is a semantic tag used for independent, self-contained content that can stand alone.

Aside

Aside is a semantic tag used for content that is related to the main content, but not part of it directly.

Footer

is a semantic tag used to define the bottom section of a webpage or section.
